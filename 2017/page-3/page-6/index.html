We offer a complete line of mud screen filters and components for down hole drilling. We manufacture mud screens in all shapes, sizes and material, using the highest quality materials. Our production line can facilitate all amounts of quantities, whether small orders or high quantity orders.

For more information about mud screens please contact us.

Our stocked screens are:

  • 3-1/2 API IF or 3-1/2 X-Hole
  • 4 API IF or 4-1/2 X-Hole
  • 4-1/2 API IF or 5 X-Hole
  • 5-1/2 API
  • 4 API FH
  • 5-1/2 FH
